# Estudo de Docker

O objetivo desse projeto é fazer um registro dos estudos na configuração de um ambiente para o desenvolvimento PHP via Docker começando do recurso mais básico, buscando reproveitar ao máximo as imagens já existentes em repositórios oficiais.

É possível que cometa alguns erros no processo e que seja necessário voltar alguns passos. Conto com críticas para o refinamento da proposta.


## Pré-requisitos

- Docker (https://docs.docker.com/install/)
- Docker Compose (https://docs.docker.com/compose/install/)


## Como rodar o projeto

Na raiz do projeto rodar o comando: `docker-compose up`
Acessar o endereço http://localhost:8000
